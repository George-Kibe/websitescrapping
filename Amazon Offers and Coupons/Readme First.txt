#How to launch the scripts manually
1. After extracting all the files, navigate to the main folder using Terminal for Mac or Command Prompt for Windows
2. After ensuring you are in the right folder, run the command "python filename.py" or by just "filename.py"

PS. Filename is the name of each file e.g. coupons.py
PSS. You must have python installed in your computer
